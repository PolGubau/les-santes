Opportunities for Enhancement for Les Santes
App
● ASO Optimization for Better Visibility
● Implementation of Dark Mode
● Dynamic Walkthrough for New Users
1) ASO Optimization for Better Visibility
Observation:
The current app description lacks comprehensive text and relevant keywords,
which is crucial for App Store Optimization (ASO).
Why It Matters:
A well-optimized app description can significantly improve the app's visibility
on app stores, helping it reach a wider audience. Effective use of keywords
can also ensure that users searching for festival-related apps are more likely to
find Les Santes.
Possible Enhancements:
● Keyword Research: Conduct thorough research to identify popular
search terms relevant to festivals and Mataró.
● Feature-Rich Description: Expand the description to include more
details about features and benefits, allowing potential users to see the
full value of the app.
● Use of Bullet Points: Summarize key features in bullet points to make
the description more skimmable and engaging.
2
2) Implementation of Dark Mode
Observation:
The app currently does not offer a dark mode feature.
Why It Matters:
Dark mode has become a popular choice for many users, particularly for
comfort during nighttime use. It helps reduce eye strain, saves battery life on
devices with OLED screens, and can enhance user satisfaction.
Possible Enhancements:
● Dark Mode Toggle: Introduce a settings option that allows users to
switch between light and dark modes easily.
● Consistent Design: Ensure that the color scheme in dark mode is
visually appealing and maintains readability, using high-contrast text
and backgrounds.
● User Preferences: Consider allowing the app to automatically switch
modes based on user device settings or time of day for added
convenience.
3) Dynamic Walkthrough for New Users
Observation:
Currently, there is no dynamic walkthrough or tutorial for new users to help
them navigate the app.
3
Why It Matters:
A guided walkthrough can significantly enhance the onboarding experience,
helping users understand key features and functionalities without feeling
overwhelmed. This can improve user retention and satisfaction.
Possible Enhancements:
● Introductory Tutorial: Develop an onboarding tutorial that highlights
major features, guiding users through initial setup and app
functionality.
● Interactive Prompts: Use interactive prompts that highlight buttons
and features as users navigate the app for the first time.
● Skip Option: Include an option for users to skip the tutorial if they
prefer to explore the app independently.

Additional Recommendations for an Enhanced
Experience
● User Feedback Channel: Implement a feedback mechanism within the
app so users can report issues or suggest improvements, fostering
community engagement.
● Enhanced Event Information: For each event, consider including
additional information such as duration, location details, and special
instructions directly in the app for better planning.
● Offline Access: Allow users to download and access the festival agenda
and relevant maps offline to enhance usability, especially in areas with
limited internet connectivity.
● Social Integration: Consider adding features that allow users to share
events or their favorite moments from the festival on social media,
which can help promote the app organically.
● Accessibility Enhancements: Ensure that the app is accessible for
users with disabilities, including screen reader compatibility and
options to adjust font sizes for better readability.
By implementing these constructive enhancements, the Les Santes app can
provide a more user-friendly, engaging, and efficient experience, making it a
valuable resource for festival-goers.
